# API DE GESTION DES DEPENSES

# Presentation
Une API complète et Sécurisée de gestion des depenses d'un utilisateur

# Fonctionnalitées
 - Authentification (Inscription et Connexion)
 - Gestion des depenses

# Techonoligies 
 - Laravel 12
 - Sanctum 
 - Redis (Caching)
 - Scramble (Documentation)
 - Pest (Tests)

# Installation
 ## Clone le projet
    ` https://github.com/TraoreAllassane10/API-GESTION-DES-DEPENSES.git `

 ## Installation des dépendances
 ` composer install `

 ## Generation de clé
 `php artisan key:generate`
